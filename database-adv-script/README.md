# Advanced SQL Joins Queries

This project is part of the **ALX Airbnb Database Module**.  
The goal is to practice writing advanced SQL queries using different types of joins.

## Files
- **joins_queries.sql** → Contains the SQL queries for INNER JOIN, LEFT JOIN, and FULL OUTER JOIN.
- **README.md** → Explains the purpose of each query.

## Queries
1. **INNER JOIN (Bookings + Users)**  
   Retrieves all bookings along with the users who made them.

2. **LEFT JOIN (Properties + Reviews)**  
   Retrieves all properties and their reviews, including properties with no reviews.

3. **FULL OUTER JOIN (Users + Bookings)**  
   Retrieves all users and all bookings, even if a user has no bookings or a booking is not linked to any user.
